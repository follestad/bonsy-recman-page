Bonsy RecMan WordPress Plugin

Save time and automatically publish job posts from RecMan to your company website. Incorporate your design and brand and let the plugin and RecMan do the rest. Fully automate the workflow and keep your website updated without additional tasks.

Download Read documentation

Bonsy RecMan WordPress Plugin will automatically publish job posts from RecMan to your WordPress site. All you have to do is to set up your design and define what you want to show. The plugin will make the integration easy, safe and fast without any knowledge about RecMan API.

Why use this Bonsy plugin for RecMan

Automate your workflow

Automatically show job posts from RecMan on your company site. No need to log into WordPress.

Full control over design

Implement your design/brand and select the fields you want to show. The plugin will handle all the technical implementation.

Built-in caching

Automatic caching will help you to not exceed RecMan API limits, as well as it keeps your site safe and fast.

SEO friendly

Get SEO friendly URL's and metadata/tags for search engines and social media sharing.

Backwards compatible

Don't worry about keeping your code updated. The plugin is backwards compatible with the RecMan API, so changes in the API will not break the site.

Keep plugin updated through WordPress

New updates to the plugin will be available through WordPress plugin updates.

Support multiple languages

Automatic translation based on selected language. Some RecMan API fields must be translated to make them readable. Currently, supports Norwegian, English and Swedish. Additional languages can be added on request.

Cleaner HTML code

The plugin will output a cleaner HTML for job posts, so it is easier to incorporate your design.

Filter by department

Want to only show job post from certain departments or corporations in RecMan? The plugin has built-in features for filtering job posts.

Supports free text search

Use the built-in search function to let visitors search through job posts.

User filter

Incorporate filters so candidates can show job posts for certain positions, locations and other fields you like to filter.

Many helper functions

Built-in helper functions to show job post and position counts, excerpt text and job post filters.

Advance filtering.

Easily create filters for all type of fields. Incorporate filters so candidates can show job posts for certain positions, locations, branches and other fields you like to filter.

Show individual job posts on your site

Show individual job posts on your site or make the list link to your RM page.

Get Started

To get started just download the plugin and upload it to WordPress. When you activate the plugin a RecMan icon will appear in the admin menu. From here you only need to register using your RecMan API key.

When you register with the API key a Bonsy account with free trial will start. From here you got 30 days to test and get things running before you need to subscribe to keep using the service.

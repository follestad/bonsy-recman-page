RecMan WordPress Plugin

Fully automate the workflow and keep your website up to date without extra work.

Download Read documentation

What does the WordPress plugin do?

The plugin will simplify the integration process of making job posts from RecMan be automatically published to your WordPress site. You only need to define which fields that you want to be displayed. The plugin will make the integration easy, safe and fast without any knowledge about the RecMan API.

Why use the plugin?

Automate your workflow

Automatically display job posts from RecMan without any manual work.

Full control over design

Implement your design and select the fields you want to display. The plugin will handle the technical part.

Built-in caching

Automatic caching will help you to not exceed RecMan API limits, as well as it keeps your site safe and fast.

SEO friendly

Will automatically use SEO friendly URLs and metadata for search engines and social media sharing.

Backwards compatible

Don't worry about keeping your code updated with API changes. The plugin is backward compatible with the RecMan API, so changes in the API will not break the site.

Keep the plugin updated via WordPress

New updates to the plugin will be available through the native WordPress plugin updater.

Multi-language support

Some RecMan API fields requires translation to make them readable. The plugin includes automatic translation in Norwegian, English and Swedish. Additional languages can be added on request.

Cleaner HTML code

The plugin will output a cleaner HTML for job posts, so it is easier to incorporate your design than the original received HTML code.

Filter by department

Want to only show job post from certain departments or corporations? The plugin has built-in features for filtering job posts.

Advanced filtering

Easily create filters for all type of fields. Incorporate filters so candidates can show job posts for certain positions, locations, branches and other fields you like to filter.

Helper functions

Built-in helper functions to show job post and position counts, excerpts, advanced filtering for users and support for freetext search.

Display individual job posts on your site

Display the content of individual job posts on your site, or make the list link directly to the RecMan page.

How to get started

Get started by downloading the plugin and upload it to WordPress. When you activate the plugin a RecMan icon will appear in the admin menu. From here you only need to add a RecMan API key.

When you register with the Recman API key a free 30 day trial will start. This gives you time to evaluate the plugin, and if you want to keep using the plugin after the 30 days, you can purchase a subscription.

Example on how simple it is to display a job post title

<h1><php the_jobpost('title');?></h1>

Or to loop through all job posts

if( have_jobposts() ) : while( have_jobposts() ) : the_jobpost();

    echo '<h1>' . get_jobpost('title') . '</h1>';

endwhile; else :

    echo 'No job posts available';

endif;

You can also use the plugin to insert job posts as WordPress posts

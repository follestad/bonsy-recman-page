Getting Started

Begin by downloading the plugin and uploading it to WordPress. Once activated, a RecMan icon will appear in the admin menu. Simply add your RecMan API key to get started.

Registering with the RecMan API key initiates a free 30-day trial, allowing you to evaluate the plugin. If you wish to continue using the plugin after the trial period, purchase a subscription.

Code example

How to display a job post field:

<h1><php the_jobpost('title');?></h1>

How to list job posts:

if( have_jobposts() ) : while( have_jobposts() ) : the_jobpost();

    echo '<h2>' . get_jobpost('title') . '</h2>';

endwhile; else :

    echo 'No job posts available';

endif;

You can also use the plugin to insert job posts as WordPress posts

Built in shortcode

There is also a built-in shortcode you can use to display the job post feed. With a simple template engine and a option to add custom CSS.

[job_post_feed]
